# Advent of Code 2023

To run the solution use: `./gradlew run [-Pday=<DAY>] [-Ppart=<PART>] [-Pexample]`.

- Day is a number between 1 and 25.
- Part is either 1 or 2. Don't specify part to run both.
- Use `-Pexample` flag to run the solution with the example input.

(If you have the whole repository, remember to run the command inside ./2023 directory `cd 2023`)

Examples:

- `./gradlew run -Pday=2` - run day 2 solution (both parts).
- `./gradlew run -Pday=5 -Ppart=1` - run day 5 part 1 solution.
- `./gradlew run -Pday=7 -Ppart=2 -Pexample` - run day 7 part 2 solution with example input.
